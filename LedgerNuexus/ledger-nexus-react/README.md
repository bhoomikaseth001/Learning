# Ledger Nexus React Application

## Overview
Ledger Nexus is a React-based application designed for analyzing Ethereum wallet transactions and detecting potential fraud. It provides users with tools to input wallet addresses, view suspicious transactions, and check for fraudulent activities using machine learning.

## Features
- **Wallet Analysis**: Analyze Ethereum wallet transactions and identify suspicious activities based on user-defined thresholds.
- **Fraud Detection**: Input transaction details to check for potential fraud using a trained machine learning model.
- **Market Overview**: Fetch and display real-time cryptocurrency market data, including the top cryptocurrencies by market cap.
- **About App**: Learn more about the application, its purpose, and its development.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/ledger-nexus-react.git
   ```
2. Navigate to the project directory:
   ```
   cd ledger-nexus-react
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm start
```
This will open the application in your default web browser at `http://localhost:3000`.

### Building for Production
To create a production build of the application, run:
```
npm run build
```
This will generate an optimized build in the `build` directory.

## Usage
- Enter your Ethereum wallet address and Etherscan API key in the Wallet Analysis tab to analyze transactions.
- Use the Fraud Detection tab to input transaction details and check for potential fraud.
- View the Market Overview tab for real-time cryptocurrency market data.
- Check the About App tab for more information about the application.
